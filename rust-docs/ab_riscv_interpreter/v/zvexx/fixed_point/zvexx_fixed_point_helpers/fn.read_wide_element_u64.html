Expand description

Read a 2*SEW-wide element as u64 from the double-width source register group of a narrowing instruction.

For narrowing instructions vs2 holds elements of width 2*SEW. The register group size is 2 * group_regs. Element i of width 2*SEW is located in the same way as a SEW-wide element of width 2*SEW (i.e., treating 2*SEW as the element width). For SEW = 32 this reads 64-bit elements; for SEW <= 16 it reads narrower elements but zero-extends to u64.

§Safety

  • 2*SEW <= 64 (Zve64x constraint: only valid for SEW <= 32; caller must verify)
  • base_reg + elem_i / (VLENB / (2*sew_bytes)) < 32
