Hashing function for at most single block worth of words using portable implementation.

This API operates on words and is GPU-friendly.

num_bytes specifies how many actual bytes are occupied by useful value in input. Bytes outside that must be set to 0.

NOTE: If unused bytes are not set to 0 or an invalid number of bytes is specified, itâ€™ll simply result in an invalid hash.
