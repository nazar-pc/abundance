Struct VariableBytes

Source 
pub struct VariableBytes<const RECOMMENDED_ALLOCATION: u32> { /* private fields */ }
Expand description

Container for storing variable number of bytes.

RECOMMENDED_ALLOCATION is what is being used when host needs to allocate memory for call into guest, but guest may receive an allocation with more or less memory in practice depending on other circumstances, like when called from another contract with specific allocation specified.
